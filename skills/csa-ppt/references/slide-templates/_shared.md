# 通用组件规范

所有模版共享以下组件规范。各模版文件中不再重复定义这些元素。

---

## 1. 通用元素规格

| 元素 | 规格 |
|------|------|
| 标题区 | `position: absolute; top: 0; left: 24pt;` 主标题 28-32pt Bold `#1a1a2e` + 副标题 14pt `#666666` |
| 标题区高度 | **58-66pt**（约 11-12%），含标题+副标题+间距 |
| 左右边距 | **18-24pt** 每侧 |
| 底部洞察栏 | 浅灰底 `#f5f5f7`，高 52-60pt，内含 2-3 列，每列有图标+标题+正文 |
| 底部栏分隔 | 左侧彩色竖线 4pt 宽（橙 `#E8913A` 或蓝 `#4472C4`） |
| 右下角标注 | 9pt `#999999` "Harness Engineering" 或项目名 |
| 内容覆盖率 | >= 95% |

---

## 2. 卡片样式

| 属性 | 值 |
|------|-----|
| 背景色 | **`#f0f2f5`**（浅灰）或 `#e8f4fd`（浅蓝，特殊强调） |
| 圆角 | **8pt** 全圆角 |
| 边框 | **无**（靠背景色差区分） |
| 阴影 | 无或极淡 `0 1px 4px rgba(0,0,0,0.05)` |
| 内边距 | 上下左右 **12-16pt** |
| 卡片间距 | **8-12pt** |

---

## 3. 序号徽章样式

| 属性 | 值 |
|------|-----|
| 形状 | 圆形（半径 14-16pt）或圆角方形（圆角 6pt） |
| 背景色 | **`#E8913A`**（主强调橙） |
| 文字 | `#ffffff` 白色，16-20pt Bold |
| 位置 | 卡片左上角，或标题左侧 |

---

## 4. 底部洞察栏样式

| 属性 | 值 |
|------|-----|
| 背景色 | **`#f5f5f7`**（比卡片更浅的灰） |
| 宽度 | 全幅（左右边距 18-24pt） |
| 高度 | **52-60pt** |
| 圆角 | **8pt** |
| 布局 | **2-3列等宽**，每列含图标(20pt) + 标题(Bold 13pt `#1a1a2e`) + 正文(11pt `#333333`) |
| 左侧竖线 | 无（底部栏整体为浅灰底，不需要竖线） |

---

## 5. 扩展区/中间公式栏样式

| 属性 | 值 |
|------|-----|
| 背景色 | **`#f0f2f5`**（浅灰） |
| 圆角 | **8pt** |
| 左侧竖线 | **4pt 宽**，蓝色 `#4472C4` 或橙色 `#E8913A` |
| 标题 | `#1a1a2e`，14-16pt Bold |
| 正文 | `#333333`，12-13pt Regular |
| 高度 | **70-85pt** |

---

## 6. 图标与装饰元素

| 属性 | 值 |
|------|-----|
| 卡片图标 | 彩色图标（橙/蓝/绿），20-24pt |
| 图表区域背景 | 浅灰 `#f0f2f5` 或浅蓝 `#e8f4fd`，圆角 8pt |
| 连接线 | 灰色 `#999999`，1-1.5pt 宽 |
| 箭头 | 对应主题色的箭头（蓝/橙/绿） |
| 彩色圆点 | 放在卡片标题左侧，直径 8-10pt，颜色对应类别 |
| 右下角标注 | `#999999`，9pt |

---

## 7. 速查 CSS 模板

```css
/* 页面基础 */
页面背景:           background: #ffffff;
卡片背景:           background: #f0f2f5; border-radius: 8pt;
特殊卡片背景:       background: #e8f4fd; border-radius: 8pt;
底部洞察栏:         background: #f5f5f7; border-radius: 8pt;
扩展区:             background: #f0f2f5; border-radius: 8pt; border-left: 4pt solid #E8913A;

/* 文字 */
主标题:             color: #1a1a2e; font-weight: bold; font-size: 30pt; text-align: left;
副标题:             color: #666666; font-size: 14pt; text-align: left;
卡片标题:           color: #1a1a2e; font-weight: bold; font-size: 16pt;
卡片引导语:         color: #E8913A; font-style: italic; font-weight: bold; font-size: 13pt;
正文:               color: #333333; font-size: 12pt;
辅助文字:           color: #666666; font-size: 11pt;
洞察栏标题:         color: #1a1a2e; font-weight: bold; font-size: 13pt;
洞察栏正文:         color: #333333; font-size: 11pt;

/* 序号徽章 */
序号徽章:           background: #E8913A; color: #ffffff; border-radius: 50%;
                    width: 28pt; height: 28pt; font-size: 16pt; font-weight: bold;

/* 列表 */
项目符号列表:       list-style: disc; color: #333333; line-height: 1.4;
```

## 8. 制作检查清单

每张幻灯片完成后，必须对照检查：

- [ ] 页面背景色是否为 `#ffffff`
- [ ] 标题是否在**左上角**，深色 `#1a1a2e` 粗体 >= 28pt，**无蓝色背景栏**
- [ ] 是否选择了合适的模版（A-F），而非所有页面同一布局
- [ ] 卡片背景是否为浅灰 `#f0f2f5`（非白色、非深色）
- [ ] **无全幅深色横条**（禁止旧版蓝色标题栏/栏头/扩展区/摘要栏样式）
- [ ] 彩色仅用于点缀（序号徽章、竖线、图标、图表），面积占比 < 15%
- [ ] 内容是否填满页面 **95%+** 面积
- [ ] 是否有底部浅灰洞察栏 `#f5f5f7`
- [ ] 圆角是否统一 **8pt**
- [ ] 最小字号是否 >= 10pt
- [ ] 每页是否包含至少一种可视化图表元素
