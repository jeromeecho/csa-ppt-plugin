每页PPT幻灯片都必须包含至少一种简单明了的可视化图表元素。这是系统级硬性规则。

**Why:** 用户反馈纯文字罗列的PPT缺乏视觉吸引力，需要图表直观展示数据和关系。

**How to apply:**
- 有数值/百分比 → 柱状图 (`pptx.charts.BAR` / `COLUMN_CLUSTERED`)
- 有占比/分布 → 饼图
- 有对比 → 分组柱状图
- 有步骤/流程 → 流程图 (SVG→PNG)
- 有层级关系 → 金字塔/同心圆
- 推荐布局: 左侧文字(35-40%) + 右侧图表(55-62%)
- 图表配色: 主系列橙 `E8913A`, 第二蓝 `4472C4`, 第三绿 `70AD47`
- 图表至少占内容区域 40% 面积
- 禁止上下堆叠文字+图表，用左右并排
